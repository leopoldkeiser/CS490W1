Creating a Github Repository

Creating a Github Account

First, we created a new account on Github.

Our First Repository

To create a Github repository, we click the "New Repository" button.

The resulting page will ask us for:

  • A project name
  • A project description
  • Whether the repository will be public or private -- NOTE: Private repositories are a paid feature of Github.
  • Whether we want to initialize the repository immediately -- for our purposes, we do!

Congratulations, we've created our first Github repository!
